Instalação e Configuração

Baixando e Instalando o Git:

  1. Acesse o site oficial do Git: https://git-scm.com/download/win.
  2. Clique no link "Download for Windows" para baixar o instalador.
  3. Após o download, execute o instalador. Isso iniciará o assistente de instalação do Git.
  4. Siga as etapas do assistente de instalação. Aqui estão algumas configurações comuns que você pode escolher durante a instalação:
  5. Complete a instalação.
  • Escolha o local de instalação. O local padrão é geralmente apropriado.
  • Escolha o editor de texto que você deseja usar para mensagens de commit. O Notepad++ é uma opção popular, mas você pode escolher outro se preferir.
  • Escolha a opção "Use Git from Git Bash only" para usar o Git Bash como sua interface de linha de comando do Git. Isso é útil para quem está familiarizado com o ambiente Unix.

Configurando o Git:

Após a instalação, você precisa configurar seu nome de usuário e endereço de email para que suas informações sejam associadas a seus commits. Siga estas etapas:

  1. Abra o Git Bash. Você pode encontrá-lo no menu Iniciar ou pesquisar por "Git Bash".
  2. Configure seu nome de usuário e email, substituindo "Seu Nome" e "seu@email.com" pelos seus próprios dados. Execute os seguintes comandos no Git Bash:

    3. git config --global user.name "Seu Nome"

    git config --global user.email "seu@email.com"


  3. Opcionalmente, você pode configurar o core.editor para definir o editor de texto padrão para mensagens de commit, se não estiver satisfeito com o editor padrão. Por exemplo, para usar o Notepad++, você pode executar o seguinte comando:

git config --global core.editor "notepad++"
